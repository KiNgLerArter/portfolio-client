#### Key-words for state update functions

- To create smth:
  - Local state - create
  - Server state - init
- To delete smth:
  - Local state - delete
  - Server state - remove
- To edit smth:
  - Local state - edit
  - Server state - update
- To retrieve smth:
  - Local state - get
  - Server state - load
